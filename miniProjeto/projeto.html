Passo 1:

Primeiro os motivos para usar o Bootstrap:


  1. Bootstrap ocupa a segunda posição como a biblioteca JavaScript mais utilizada;
  2. O Bootstrap segue o conceito mobile-first;
  3. O padrão visual de estilização segue as tendências de design utilizadas no momento;
  4. Sua utilização elimina a necessidade de escrever muitas linhas de código e, ainda, contribui para reduzir o tamanho dos arquivos trafegados;
  5. E, por ser um framework de código aberto, o Bootstrap conta com uma comunidade ativa de desenvolvedores.
Passo 2:

Agora precisa baixar o Bootstrap:


Existem diferentes formas de baixar o framework. Uma delas é fazer o download da versão compilada por meio da página do Bootstrap.

Há também a opção de baixar o código-fonte do framework.

E quem não quiser baixar os arquivos pode ter acesso ao framework sem a necessidade de instalá-los no servidor. Para isso, é só utilizar os links para acessar o BootstrapCDN e, dessa forma, adicionar as referências para os arquivos necessários para a sua utilização.

Passo 3:

Pronto, já pode usar!


Adicione os arquivos “bootstrap.min.css” e “bootstrap.min.js” nas pastas correspondentes a eles dentro do seu site.

Depois, é só chamar o CSS e o JS dentro da tag do HTML.

É importante lembrar que este framework acessa a biblioteca jQuery para ativar o comportamento de seus plugins. Portanto, lembre-se de inseri-lo em sua página para que tudo funcione perfeitamente.

Layout de Grid responsivo

Bom, basicamente, o Bootstrap funciona no sistema de grids, com linhas e colunas. Cada linha pode possuir até 12 colunas. E, essas linhas, devem estar dentro de containers e as colunas dentro de linhas.

Row – No Bootstrap as linhas são identificadas por div’s com a classe ‘row’. / Col – As colunas são identificadas por div’s com classes ‘col’ e suas variações;

Containers – As linhas devem estar dentro de containers que, também, são div, representado pelas classes: ‘container’ e ‘container-fluid’;

O Bootstrap possui 5 breakpoins responsivos, ou seja, vc pode modelar o layout considerando até 5 tamanhos de telas diferentes (Extra-small/Small/Medium/Large/Extra-large).

Margin-Padding

Servem para controlar como elementos e componentes são espaçados e dimensionados. Bootstrap 4 possui uma escala de cinco níveis para utilitários de espaçamento, baseado no valor padrão 1rem da variável $spacer.

Escolha valores para todas viewports (.mr-3, no caso margin-right: 1rem) ou variações responsivas para mirar em viewports específicas (.mr-md-3 para margin-right: 1rem em viewports do breakpoint md).

Para mais informações confira a documentação.

Cores-Bordas

Transmita mensagens através de cores e estilize rapidamente bordas, com as classes:

  • class="text-primary" (secondary, success, danger warning, info, light, dark, body, muted, white, black);
  • class="bg-primary" (secondary, success, danger, warning, info, light, dark, white, transparent);
  • class="border" (top, right, bottom, left);
  • class="border border-primary" (secondary, success, danger warning, info, light, dark, white).
Buttons

Use os botões Bootstrap personalizados para ações em formulários, diálogos e outras coisas. Lembrando que esse componente tem suporte a escolha de tamanhos, estados e muito mais. Exemplos:

  • class="btn btn-primary" (secondary, success, danger warning, info, light, dark, link);
  • class="btn btn-outline-primary (secondary, success, danger, warning, info, light, dark);
  • class="btn btn-primary btn-lg" (sm);
  • class="btn btn-primary btn-lg active" (ou disabled).

Inputs

Alguns exemplos do uso de inputs:

  • Coloque um add-on ou botão em qualquer um dos lados de um input;
  • Use as classes de dimensionamento de formulário no .input-group e conteúdos dentro dele vão, automaticamente, redimensionar;
  • Coloque qualquer checkbox ou radio dentro do addon de grupo de input, invés de texto;
  • Apesar de múltiplos inputs serem suportados (visualmente), estilos de validação só estão disponíveis para grupos de input com um único input;
  • Múltiplos add-ons são suportados e podem ser misturados com input checkbox e radio;
  • Input de arquivo personalizado;
  • Botões com dropdowns, seguimentados, formulários, select, entre outros.


Facilmente, extenda os controles de formulário com textos, botões, selects personalizados, etc.

Algumas clases:

  • class="input-group";
  • class="input-group-prepend" (ou append);
  • class="input-group-text".
